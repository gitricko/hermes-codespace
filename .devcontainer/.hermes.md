# Memory Automation

## Before Every Turn
Auto-recall mnemon for the topic. If hook returns nothing, run `mnemon recall "<topic>" --limit 10` manually.

## After Every Response — Two-Tier Save
**1–2 simple facts:** Direct `mnemon_remember()` tool call (zero token overhead).
**3+ items or complex extraction:** Delegate to subagent with `toolsets=["terminal"]` (worth ~20K tokens for bulk).
When in doubt, save it — just use the right tier.

## What NOT to Save
Code in git, public docs, transient state, things already in this file.
